# My website.
## Site: https://mokon1.github.io/

### Technologies:
- HTML
- SASS
- jQuery
- and built with gulp.js (sass, autoprefix, babel).

### Features:
- sticky menu
- hamburger menu
- smooth scrolling
- animation in CSS e.g. typewriter effect
- media queries/mobile first
- loading sign
- sound- typewriter effects
